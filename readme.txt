Создание базы.
(для соединения используется localhost:5432)

1) Используя существующее соединение, создадим нового пользователя и базу
CREATE ROLE dev WITH PASSWORD 'root';
ALTER ROLE dev WITH LOGIN;
CREATE DATABASE test OWNER dev;

2) В базе test создаем схему dev
CREATE SCHEMA IF NOT EXISTS dev AUTHORIZATION dev;

3) В схеме dev создаем таблицу notes
CREATE TABLE dev.notes (	
	id INTEGER NOT NULL GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
	header VARCHAR(200),
	body VARCHAR(32767),
	creation_date TIMESTAMP
);
